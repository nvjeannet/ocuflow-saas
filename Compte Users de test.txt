Lancer le serveur:
# Depuis la racine du projet :
   npm run dev
# Ou depuis le dossier backend :
   node index.js

Ouvrez votre navigateur sur http://localhost:3000

Pour le déploiement, il vous reste à :
1-Mettre les vrais secrets dans .env de production (openssl rand -hex 32)
2-Configurer Nginx + SSL (Let's Encrypt)
3-Installer PM2 : npm install pm2 -g && pm2 start index.js --name ocuflow
4-Exécuter npm run migrate sur le serveur
